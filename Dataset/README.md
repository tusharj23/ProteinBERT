This folder contains scripts for downloading and preprocessing protein datasets used in the project.

### Files

* **download_tape_data.py** – Downloads benchmark datasets from the TAPE library.
* **preprocess_data.py** – Converts protein sequences into tokenized numerical format for model input.

### Requirement

Install the required library:

pip install tape-proteins

### Note

Datasets are not stored in this repository due to their large size.
Use the provided scripts to download them automatically.
